


# 🎬 YouTube Playback Manager



> 🎯 A smart browser extension that manages YouTube playback across multiple tabs for a seamless viewing experience.



## ✨ Features

- 🎮 **Auto-Pause Other YouTube Tabs**  
  When a video starts playing, all other YouTube videos in separate tabs are automatically paused.
  
- ▶️ **Auto-Play on Active Tab**  
  Resume video playback automatically when a tab becomes active.

- ⏸️ **Pause When Inactive**  
  Automatically pauses YouTube playback when switching away from the tab.

- 🌙 **Silent Hours**  
  Schedule time ranges (e.g., midnight–8AM) where auto-pause is disabled — ideal for sleep playlists.

- 📋 **Channel Whitelist**  
  Add specific channels that won't be affected by the auto-pause logic.

- 🔊 **Volume Normalization**  
  Keep playback volume consistent across all YouTube videos.

- 🌓 **Dark/Light Mode Toggle**  
  Select your preferred popup interface theme.

- 🔄 **Sync Settings Across Devices**  
  Automatically sync preferences using your browser’s native sync features.

- ⌨️ **Keyboard Shortcut**  
  Use `Ctrl + Shift + Y` to toggle the extension globally.

## 📥 Installation

### 🧩 Chrome / Microsoft Edge

1. Download the latest version from the [Releases](https://github.com/inyogeshwar/youtube-playback-manager/releases)
2. Extract the ZIP
3. Open your browser and go to `chrome://extensions/`
4. Enable **Developer Mode**
5. Click **Load unpacked** and select the extracted folder

### 🦊 Firefox

Available on the Firefox Add-ons marketplace:  
👉 [YouTube Playback Manager on Firefox](https://addons.mozilla.org/en-US/firefox/addon/youtube-playback-manager/)

## 🔒 Privacy Policy

This extension respects your privacy:

- ✅ Only runs on YouTube (`youtube.com`)
- ❌ No tracking or analytics
- ❌ No data is sent to external servers
- 🔐 All settings are stored locally or synced via Chrome/Firefox sync
